## 🎮 핵심 기능 시연 (영상의 메인 파트)

### 3. 플레이어 조작 및 이동 (20-30초)
- [ ] **부드러운 이동** 시연
  - WASD 이동을 자연스럽게 보여주기
  - 다양한 방향으로 움직이기
- [ ] **대시 기능** 시연
  - 빠른 이동과 회피 보여주기
  - 쿨다운 시스템 언급 가능
- [ ] **캐릭터 애니메이션** 강조
  - Idle, Walk, Run 애니메이션 보여주기

**촬영 팁:**
- 카메라가 플레이어를 부드럽게 따라가는 것 보여주기
- 배경과 환경 디테일도 함께 보이도록

### 4. 기본 전투 시스템 (30-40초)
- [ ] **기본 공격** 시연
  - 여러 각도에서 공격하기
  - 공격 애니메이션 강조
- [ ] **검기 이펙트** 보여주기
  - SlashEffect가 잘 보이도록
- [ ] **적 AI** 상호작용
  - 적이 플레이어를 추적하는 모습
  - 전투 중 긴장감 있는 장면
- [ ] **피격 및 넉백** 시스템
  - 플레이어 피격 모션
  - 적 피격 모션

**촬영 팁:**
- 다수의 적과 전투하여 역동성 강조
- 체력바 감소를 명확히 보여주기

### 5. 궁극기 시스템 - Blade Dance (30-40초) ⭐
- [ ] **궁극기 게이지 충전** 과정 보여주기
  - 공격하면서 게이지가 차는 모습
- [ ] **궁극기 발동 연출**
  - 발동 순간을 슬로우 모션으로 (편집)
- [ ] **6방향 검기** 효과 강조
  - 모든 방향의 검기가 보이도록 각도 조절
  - 이펙트 애니메이션 강조
- [ ] **다수의 적 처치** 장면
  - 궁극기로 여러 적을 한 번에 처치

**촬영 팁:**
- 이 부분이 가장 화려하므로 충분한 시간 할애
- 여러 각도에서 촬영하여 가장 멋진 장면 선택

### 6. 무기 티어 시스템 (20-30초)
- [ ] **다양한 티어의 무기** 보여주기
  - 티어 1 → 티어 3 → 티어 5 순서로
- [ ] **공격 범위 변화** 시연
  - 티어별 공격 범위 차이 비교
- [ ] **검기 색상 변화** 시연
  - 티어별 색상 변화 강조

**촬영 팁:**
- 화면을 분할하여 비교 영상으로 편집 가능
- "Tier 1" "Tier 5" 텍스트 오버레이 추가

### 7. 맵 및 탐험 시스템 (25-35초)
- [ ] **미니맵 기능** 시연
  - 미니맵 토글 보여주기
  - 플레이어 위치 추적
- [ ] **전체 맵** 열기
  - 탐험한 지역 표시
  - NPC 위치 표시
- [ ] **Quest 가이더** 기능
  - 목표 지점으로 안내받는 모습
- [ ] **다양한 맵** 탐험
  - Village → Forest → Cave 등 이동

**촬영 팁:**
- 맵이 점차 밝아지는(탐험되는) 모습 강조
- 각 맵의 분위기와 디자인 보여주기

### 8. NPC 상호작용 및 다이얼로그 (20-30초)
- [ ] **NPC와 대화** 시연
  - 다이얼로그 시스템 보여주기
  - ThaleahFat 폰트 강조
- [ ] **퀘스트 수락** 과정
  - NPC_Puangi와의 상호작용
- [ ] **스토리 전달** 방식
  - 게임의 스토리텔링 간략히 보여주기

**촬영 팁:**
- 다이얼로그 텍스트가 잘 읽히도록 속도 조절
- 자막 추가 고려

### 9. 인벤토리 및 Drag & Drop 시스템 (20-30초) ⭐
- [ ] **인벤토리 UI** 열기
  - 깔끔한 UI 디자인 강조
- [ ] **드래그 앤 드롭 기능** 시연
  - 아이템을 드래그하여 이동하는 모습
  - 부드러운 인터랙션 강조
  - 아이템 스왑(교환) 보여주기
- [ ] **아이템 획득** 장면
  - 드롭 아이템 줍기
- [ ] **아이템 사용** 시연
  - 소비 아이템 효과
- [ ] **무기 장착** 시연
  - 장착 시 스탯 변화

**촬영 팁:**
- 드래그 앤 드롭이 실제로 작동하는 모습을 명확히 보여주기
- 마우스 커서가 잘 보이도록 설정
- 여러 아이템을 이동시켜 시스템의 완성도 강조

### 10. 세이브/로드 시스템 (10-15초)
- [ ] **세이브 UI** 보여주기
  - 여러 세이브 슬롯
  - 세이브 정보 표시 (시간, 위치 등)
- [ ] **로드 기능** 시연
  - 빠른 로드
  - 게임 상태 복구 확인

### 11. 보스전 (40-60초) ⭐
- [ ] **보스 등장** 연출
  - 보스 체력바 등장
  - 긴장감 있는 분위기
- [ ] **보스 전투** 시연
  - 다양한 보스 패턴 보여주기
  - 플레이어의 회피 및 공격
  - 궁극기 사용 타이밍
- [ ] **Unkillable Boss** 특별 시연
  - 죽일 수 없는 보스 컨셉 보여주기
- [ ] **보스 처치** 연출
  - 승리 모션

**촬영 팁:**
- 보스전은 게임의 하이라이트이므로 충분한 시간 할애
- 긴박감 있는 장면 선택
- BGM이 잘 들리도록 오디오 믹싱

### 12. 시네마틱 및 스토리 (20-30초)
- [ ] **튜토리얼 시네마틱** 일부 보여주기
- [ ] **Sad Scene** 영상 일부 (스포일러 주의)
- [ ] **Ending Scene** 티저 (결말은 감추기)

**촬영 팁:**
- 전체를 보여주지 말고 분위기만 전달
- "스토리가 있는 게임"임을 강조

### 13. Master UI & 설정 시스템 (15-20초) - 기술적 하이라이트
- [ ] **여러 UI 창 전환** 보여주기
  - 인벤토리 → 맵 → 퀘스트 등 빠른 전환
  - UI가 겹치지 않고 관리되는 모습
- [ ] **설정 패널 (Settings)** 시연
  - **사운드 조절**: BGM/효과음 볼륨 슬라이더
  - **밝기 조절**: 화면 밝기 조정 기능
  - 설정 변경이 즉시 반영되는 모습
- [ ] **ESC키로 UI 닫기** 연출
  - 직관적인 UI 관리 시스템 강조

**촬영 팁:**
- 사운드 조절 시 실제로 볼륨이 변하는 것 보여주기
- 밝기 조절 전/후 화면 차이 명확히 표현
- "Master UI Controller" 텍스트 오버레이

### 14. 맵 생성 시스템 (15-20초) - 기술적 하이라이트

- [ ] **Ninja Village Generator** 결과물
  - 절차적 생성 맵의 다양성

**촬영 팁:**
